# ShopEasy — Магазин одежды и продуктов

Мобильное Android-приложение для просмотра каталога товаров (одежда и продукты), 
формирования корзины и оформления заказа.

## Возможности
- Авторизация / регистрация через Firebase Auth
- Каталог товаров с поиском и фильтрацией по категориям
- Корзина (добавление, изменение количества, удаление)
- Избранное
- Добавление новых товаров
- Данные сохраняются локально (Room Database)
- Система уведомлений

## Технологии
- Kotlin + Jetpack Compose
- Room Database
- Firebase Auth
- Navigation Compose
- MVVM-архитектура
- Coil (загрузка изображений)

 Как запустить
1. Клонируйте репозорий
2. Добавьте `google-services.json` в папку `app/`
3. Откройте проект в Android Studio
4. Запустите на эмуляторе (API 34)
